Daftar Program Studi

Teknik Informatika

Program studi Teknik Informatika fokus pada pengembangan perangkat lunak, jaringan komputer, dan sistem informasi. Lulusan diharapkan mampu menciptakan solusi teknologi inovatif.

Detail Program

Sistem Informasi

Program studi Sistem Informasi membekali mahasiswa dengan keahlian dalam perancangan, pengembangan, dan manajemen sistem informasi untuk mendukung operasional bisnis.

Detail Program

Sistem Komputer

Program Studi Sistem Komputer Universitas Handayani Makassar mempersiapkan mahasiswa dengan keahlian mendalam dalam bidang perangkat keras komputer, arsitektur sistem, jaringan, dan pengembangan sistem tertanam (embedded systems). Lulusan akan mampu merancang, membangun, dan mengelola berbagai sistem komputasi.
